# **App Name**: Veltrix

## Core Features:

- Hero Section: Display a hero section with a catchy headline, subheadline, and a 'Get Started' CTA that scrolls to the pricing section.
- Pricing Plans: Implement a pricing section displaying subscription plans (1, 6, and 12 months) in a card layout with 'Buy Now' buttons. Implement placeholder Stripe/Paypal
- Features Showcase: Showcase key benefits of the IPTV service (HD/4K, No buffering, Global access, Compatibility, Support).
- FAQ Section: Provide an FAQ section with answers to common questions about IPTV subscriptions, compatibility, etc.
- Contact Form: Enable a contact form (Name, Email, Message) with company email and location placeholders.

## Style Guidelines:

- Light background (#F9FAFB or white) with soft shadows for a clean look.
- Google Blue (#4285F4) as the primary accent color for interactive elements.
- Use either Inter or Roboto font for a modern and readable text.
- Utilize rounded corners and card layouts inspired by Google's Material Design.
- Incorporate hover effects and smooth transitions for a seamless user experience.
